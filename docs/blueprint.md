# **App Name**: DocuMind

## Core Features:

- Smart Document Upload: Multi-format drag-and-drop area for PDF, DOCX, PPTX, and TXT files for knowledge ingestion.
- Intelligent Retrieval Agent: An AI tool that reasons across your documents to find exact information using semantic search.
- Citation & Source Linking: Display page numbers and specific document excerpts alongside AI responses for verification.
- Predictive Follow-ups: Generate dynamic follow-up question tools based on the current chat context to guide discovery.
- Knowledge Cube Dashboard: A centralized visual repository displaying all uploaded source files for quick navigation.
- Cross-Document Synthesis: An AI-powered feature to summarize or compare data across multiple different files simultaneously.

## Style Guidelines:

- Primary color: Electric Indigo (#6366F1), evoking intelligence and premium tech.
- Background: Minimalist Lavender White (#FBFBFF), providing a clean, airy canvas for content.
- Accent: Deep Azure (#3B82F6), used for active states and critical user actions.
- Headline font: 'Poppins' for a contemporary and avant-garde feel. Body font: 'Inter' for neutral, machining-style readability in chat threads.
- Line-based icons with subtle rounded edges and varying stroke weights for a sophisticated Apple-inspired look.
- A focused three-column chat layout maximizing whitespace, with documents on the left, conversation center, and sources on the right.
- Framer Motion transitions for documents sliding into the 'AI Cube' and message bubbles entering with smooth opacity fades.
